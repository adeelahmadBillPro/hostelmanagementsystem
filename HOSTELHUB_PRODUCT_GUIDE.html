HostelHub - Complete Hostel Management Platform

Product Overview

is a comprehensive, cloud-ready hostel management SaaS platform built for hostel owners, managers, and residents in Pakistan. It digitizes every aspect of hostel operations - from room management to billing, food ordering, staff tracking, and resident portals.

Next.js 14 | React 18 | PostgreSQL | Prisma ORM | Tailwind CSS | TypeScript

Security & Validation

  • on all 11+ API endpoints
  • Phone: Pakistani format, repeated digit rejection
  • CNIC: 13-digit validation, duplicate check, fake pattern detection
  • Email: Format + disposable domain blocking
  • Password: 8+ chars, uppercase, lowercase, number, special character
  • Amount: Min/max bounds, NaN check
  • Rate limiting on all APIs (standard, sensitive, strict)
  • Role-based access control (5 roles)
  • JWT session management
  • bcrypt password hashing
